# Kown — 公共分发

[Kown](https://github.com/macuhy/kown) 的公共发布仓库：存放各版本的下载产物（Releases）与 Sparkle 自动更新源（`appcast.xml`）。

- **下载**：见本仓库 [Releases](https://github.com/macuhy/kown-mac/releases)。
- **自动更新**：App 内置 Sparkle，按 `appcast.xml` 检查并安装更新。

> 本仓库内容由 `macuhy/kown` 的发布流程（GitHub Actions）自动更新，请勿手动编辑 `appcast.xml`。
